# Example 10: Showcase

A filled 3D heart with interior voxels and hierarchical tree structure visualization - perfect for demonstrating GPU-VDB's sparse storage efficiency.

## What it does

Generates a solid heart shape (not just surface) with height-based gradient coloring, showing how GPU-VDB efficiently stores thousands of voxels using a hierarchical tree structure (Blue=Root, Green=Internal, Red=Leaf nodes).
